# 🌳 Soy Ağacı

Aile soy ağacı oluşturma ve yönetme uygulaması.

## Özellikler
- Kişi ekleme, düzenleme, fotoğraf yükleme
- İlişki tanımlama (eş, ebeveyn, kardeş, amca/dayı/hala/teyze, büyükanne/baba)
- Ağaç görselleştirme (sürükle/bırak, pinch-to-zoom)
- Proje kaydetme, dışa/içe aktarma
- Yazdırma ve PDF kaydetme
